# 🚀 AI CRM Dashboard (Frontend)

A modern, enterprise-grade **Customer Relationship Management (CRM) Dashboard** built using **React, Vite, and JavaScript**. This project replicates the user experience of popular CRM platforms like **Salesforce, HubSpot, Zoho CRM, and Monday.com**.

The application is designed with a scalable architecture, reusable components, responsive layouts, and a service layer that can be connected to a real **Node.js + Express.js + MongoDB (MERN)** backend without changing the UI.

> **Project Type:** Frontend Only
> **Architecture:** MERN Ready (Frontend completed, Backend can be integrated later)

---

# 📖 Project Overview

The AI CRM Dashboard helps businesses manage their daily operations from a single platform.

It provides modules for:

* Customer Management
* Lead Management
* Product Management
* Order Management
* Employee Management
* Business Reports & Analytics
* AI Assistant Interface
* Notifications
* Company Settings

The goal of this project is to demonstrate how a real-world SaaS CRM application is built using modern React development practices.

Install dependencies:


Start the development server:

```bash

```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
